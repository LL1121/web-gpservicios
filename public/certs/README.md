# Certificados PDF

Soltá acá los PDFs reales con estos nombres exactos:

- `iso-9001.pdf`
- `iso-14001.pdf`
- `iso-45001.pdf`
- `sgi-politica.pdf`

Si querés sumar / cambiar certificados, editá `lib/certifications.ts` con el nuevo
`slug`, `fileUrl` y `verifyUrl`.

Mientras no estén los archivos reales, el visor del sitio mostrará un mensaje
"no se puede previsualizar" — pero el botón **Verificar** sigue funcionando
porque apunta al organismo emisor (IRAM, etc.).
